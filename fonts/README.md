# Fonts

Fonts are loaded via Google Fonts (no local TTFs bundled).

## Arabic (primary)
- **Tajawal** — modern sans with strong legibility, excellent for UI body. Weights 300/400/500/700/900.
- **Amiri** — classical Naskh for Qur'anic passages, hadith, display/ceremonial. Weights 400/700, italic available.
- **Reem Kufi** — geometric Kufi for display/institutional lockups. Weights 400–700.

## Latin (secondary)
- **Inter** — neutral sans used when English appears alongside Arabic body.
- **Cormorant Garamond** — serif display paired with Amiri for English subtitles.

## Substitution note
No proprietary foundry fonts exist yet for the brand. When real typefaces are licensed, drop the TTF/WOFF2 files into this folder and update `colors_and_type.css` `@font-face` blocks.
