# Public site — donor & registration

Marketing site that introduces the madrasa, channels donations, and opens two registration flows (boys, girls).

## Screens
- **Homepage** — hero with hero quote, stats strip, intro, registration CTAs, campaigns with progress bars, footer.
- **Registration — boys** — parchment-themed, three form sections (student / guardian / notes).
- **Registration — girls** — mirror flow, identical tone.
- **Success state** — after submission, shown with 8-point star.

## Components demonstrated
- Sticky translucent header with brand lockup
- Gold progress bars
- Campaign cards with girih-patterned imagery
- Parchment form pages (light theme via `[data-theme="paper"]`)
- Honoured address forms (الأهل الكرام) following the content rules

## Entry
`index.html` — single-file, no external JS dependencies beyond the shared `colors_and_type.css`.
